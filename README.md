## Getting Started

```bash
git clone https://github.com/alopezprlp/kidsdev.git

cd kidsdev

npm install

npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
